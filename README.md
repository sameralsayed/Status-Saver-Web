# Status Saver Web

💾 Status Saver Web - Save WhatsApp status images and videos instantly with preview and offline gallery.

## ✨ Features
- 📸 **Images Tab** — Browse and save status photos
- 🎥 **Videos Tab** — Preview and save status videos with player
- 💾 **Saved Tab** — View, play, and manage all your saved statuses
- 👁️ **Preview Before Saving** — Click to view full size or play video
- ⬇️ **One-Tap Download** — Save directly to browser storage
- 🗑️ **Delete Saved Items** — Clean up your gallery easily
- 📱 **Mobile-First Design** — Perfect experience on phones and tablets
- 🔒 **100% Private** — All data stored locally in your browser
